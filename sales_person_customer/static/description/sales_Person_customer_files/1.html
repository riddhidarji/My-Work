Mass Update Salesperson

Select the customer from the list view and then from the "Action" menu select "Mass Update".

After that one wizard opens, you can update multiple customer's salesperson from here.

Salesperson Updated.

Update it by 2 methods, "Add" and "Replace".

Add: It add a new salesperson in the old salesperson. 

Allocated salesperson Updated.

Update salesperson by using the method "Replace".

Replace: It replace new salesperson with old salesperson , it removes old salesperson.
